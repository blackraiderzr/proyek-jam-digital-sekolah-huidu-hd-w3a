# Presentasi Kelompok 1 - Proyek Jam Digital Sekolah

## Identitas Kelompok
- **Sekolah:** SMKN 9 SURAKARTA
- **Jurusan/Kelas:** XI TJKT 3
- **Kelompok:** 1
- **Topik:** Implementasi Jam Digital Sekolah berbasis Huidu HD-W3A

## Tujuan Proyek
1. Menampilkan jam dan tanggal real-time pada panel LED sekolah.
2. Menyajikan informasi sekolah secara ringkas dan jelas.
3. Membangun media pembelajaran IoT/display control untuk siswa.

## Perangkat & Software
- Controller: **Huidu HD-W3A**
- Software konfigurasi: **HD2020**
- Panel LED: P10/P5 (disesuaikan kebutuhan)
- Koneksi: WiFi / LAN

## Alur Implementasi
1. Menentukan ukuran panel LED.
2. Menyusun layout jam, tanggal, dan teks identitas.
3. Mengatur sinkronisasi waktu (NTP) pada HD2020.
4. Upload konfigurasi ke controller HD-W3A.
5. Uji tampilan dan optimasi kecerahan.

## Hasil yang Diharapkan
- Tampilan waktu akurat dan mudah terbaca.
- Informasi identitas sekolah tampil rapi.
- Sistem stabil untuk penggunaan harian di lingkungan sekolah.
